# Privacy Policy — Sleeping Bird

**Last updated:** March 14, 2026

## Overview

Sleeping Bird is a Chrome extension that helps users generate AI-powered reply suggestions on X.com (formerly Twitter). Your privacy is important to us.

## Data Collection

This extension does **not** collect, store, or transmit any personal data to our servers. We do not have servers.

### What we store locally

- **Grok API Key:** Your API key is stored in Chrome's synced storage (`chrome.storage.sync`) so it persists across sessions. It never leaves your browser except when sent directly to the Grok API to generate replies.

### What we send to third parties

- **Grok API (api.x.ai):** When you click "Generate," the text of the tweet you're replying to and your draft reply are sent to the Grok API to generate suggestions. This data is governed by [xAI's privacy policy](https://x.ai/legal/privacy-policy).

### What we do NOT collect

- We do not collect browsing history, personal information, analytics, or telemetry.
- We do not track your usage of X.com.
- We do not store tweet content or generated replies.
- We do not use cookies or any tracking mechanisms.

## Permissions

- **storage:** Used solely to save your API key locally.
- **host_permissions (x.com, twitter.com):** Required to inject the reply assistant UI into X.com pages.

## Data Security

Your API key is stored using Chrome's built-in encrypted storage. All API communication uses HTTPS.

## Changes to This Policy

We may update this policy from time to time. Changes will be reflected in the "Last updated" date above.

## Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
